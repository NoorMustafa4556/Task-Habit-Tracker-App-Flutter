# User Stories

1. As a new user, I want to create an account so my progress is tracked.
2. As a returning user, I want to log in securely to access my habit list.
3. As a user, I want to see a list of habits on my home screen.
4. As a user, I want to see a detailed view of each habit to understand its status.
5. As a user, I want to see a daily motivational quote to stay inspired.
6. As a user, I want my login state to be remembered even after the app closes.
7. As a user, I want to toggle notifications in settings to manage reminders.
8. As a user, I want to see an error message if I enter wrong credentials.
9. As a user, I want to log out safely to protect my privacy.
